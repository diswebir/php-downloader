# دانلودر ساده PHP

دانلودر تک‌فایلی با رابط کاربری ساده (RTL) برای دانلود فایل از لینک‌های http/https مستقیماً روی هاست. دارای نوار پیشرفت، سرعت، ETA و فونت فارسی Vazirmatn.

- سبک و بدون وابستگی خارجی (به جز cURL و فونت از Google Fonts)
- نمایش پیشرفت، حجم دانلود شده/کل، سرعت و زمان باقی‌مانده
- ذخیره‌سازی فایل‌ها در پوشه `downloads`
- جلوگیری از تکرار نام فایل (auto unique)
- دنبال کردن ریدایرکت‌ها
- پاکسازی خودکار پوشه موقت `tmp` پس از اتمام (از طریق endpoint تمیزکاری)
- فایل‌های `.htaccess` برای امنیت پوشه‌ها (Apache)

## پیش‌نیازها
- PHP 7.2 یا بالاتر
- افزونه cURL فعال در PHP
- دسترسی نوشتن روی مسیر اسکریپت برای ساخت پوشه‌های `downloads` و `tmp`
